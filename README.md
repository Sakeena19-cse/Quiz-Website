# Quiz Website 

A simple and interactive **Quiz Website** built using **HTML, CSS, and JavaScript**. This project contains 5 basic quiz questions and is designed to help beginners understand core front-end web development concepts.


##  Features

* 5 basic quiz questions
* Interactive user interface
* Dynamic question handling using JavaScript
* Score calculation
* Simple and clean design

##  Technologies Used

* **HTML** – Structure of the website
* **CSS** – Styling and layout
* **JavaScript** – Quiz logic and interactivity

##  How to Run the Project

1. Open the project folder
2. Open the `index.html` file in your web browser
